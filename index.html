Introduction

React is a JavaScript library for building user interfaces. React has been designed from the start for gradual adoption, and you can use as little or as much React as you need. Whether you want to get a taste of React, add some interactivity to a simple HTML page, or start a complex React-powered app, the links in this section will help you get started.

Hello World

The smallest React example looks like this:

ReactDOM.render( <h1>Hello, world! </h1>, document.getElementById('root'));

Handling Events

Handling events in React elements is very similar to handling events in DOM elements. There are some syntax differences:

  • React events are named using camelCase instead of lowercase letters.
  • With JSX you pass a function as an event handler instead of text.

Lists and Keys

Lists are an important aspect of your application. Every application is required to make use of lists in one way or another. Below is an example of a list in React:

function NumberList(props) {
const numbers = props.numbers;
const listItems = numbers.map((number) => < li key={number.toString()}> {number} < / li> );
return ( < ul > {listItems} < / ul> );
}
